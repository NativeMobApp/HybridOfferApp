Aclaraciones antes de instalar flutter:
1. Instalar Flutter SDK (yo tengo la versión: Flutter 3.38.1 y Dart: Dart 3.10.0 • DevTools 2.51.1)

2. Conifgurar las variables de entorno para que el windows pueda reconocer los comandos de flutter (Si no lo hiciste, anda al min 39;40: https://www.youtube.com/watch?v=IKG1eV2SetA&t=12074s)

3. Clonar el repo HybridOfferApp en una carpeta local (ya viene el todo el código nativo integrado con el flutter)

Para desarrollar en Flutter:

  a. Abrir VS Code en la carpeta `flutter_module_hybrid` del nuevo repo (abrir cmd en esa carpeta y poner el comando: code)
    
  b. Ejecutar: en el mismo cmd poner el comando: flutter pub get y no cerrar.
    
  c. En VS CODE, selecciona el archivo `main.dart` y presionar F5
     
  d. Si tenes ya configurado el autoguardado de vs code, se va a autocompilar cada vez que flutter detecta un cambio
